# Restaurant-Website
A static restaurant website built on html5, css3 and bootstrap 3.3.6. This website has a responsive based UI\UX. It has two web pages home and menu pages and google map of the restaurant is also embedded in the website. 
The Home contains the big-restraunt picture below it the three tiles menu-tiles which redirects user to the menu page containig all the menu-items,specials-tiles it again redirects the user to the menu page which containing all the specials-item and the third tiles is the restraunt map
